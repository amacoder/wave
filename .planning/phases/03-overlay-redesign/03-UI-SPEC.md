# Phase 3 — UI Design Contract

> Visual and interaction contract for Phase 3: Overlay Redesign.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

## Phase UI Surface

Phase 3 is a **full rewrite of `RecordingOverlayView.swift`** plus a targeted timing change in `AppDelegate.swift`. It touches exactly one UI surface: the floating pill overlay. No settings screens, menus, or new windows are introduced.

The surface area:
- `RecordingOverlayView.swift` — rewritten top-to-bottom: pill shape, four-state ZStack, Canvas waveform, phase-gated animations
- `AppDelegate.swift` — done-state hide timing moved from immediate to +0.8s asyncAfter

Source: RESEARCH.md — "No new files needed. No other files touched."

## Spacing Scale

The overlay pill uses a compact spacing contract. All values are multiples of 4.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4pt | Gap between waveform bars (2pt gap is a Canvas drawing value, not a layout token) |
| sm | 8pt | HStack inter-element gap within state views |
| md | 16pt | Horizontal padding inside the pill HStack |
| lg | 24pt | Not used in the pill itself; inherited for any future surfaces |

**Pill geometry (fixed dimensions — not spacing tokens):**

| Dimension | Value | Source |
|-----------|-------|--------|
| Pill width | 280pt | RESEARCH.md Pattern 2 |
| Pill height | 52pt | RESEARCH.md Pattern 2 |
| Bottom margin | 32pt above Dock (`screenFrame.minY + 32`) | RESEARCH.md Pattern 2 |
| Waveform Canvas width | 80pt | RESEARCH.md Pattern 4 |
| Waveform Canvas height | 24pt | RESEARCH.md Pattern 4 |
| Recording indicator circle (outer) | 32pt diameter | default — proportional to 52pt pill height |
| Recording indicator dot (inner) | 12pt diameter | default — proportional to outer |

Exceptions: Touch targets not relevant — overlay is `ignoresMouseEvents = true`. No interactive hit targets on the overlay surface.

## Typography

The pill shows brief status labels only. No headings or display text.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Label (status) | 13pt | Semibold (600) | 1.2 | "Recording...", "Transcribing..." inside pill |
| Caption | 11pt | Regular (400) | 1.2 | "ESC to cancel" hint badge |

> Two sizes, two weights. Line height 1.2 is appropriate for single-line labels on a compact pill — no multi-line wrapping occurs.

Source for sizes: Existing RecordingOverlayView.swift uses `.font(.headline)` (17pt) for status labels — Phase 3 downsizes to `.font(.subheadline)` (13pt semibold) to match a compact 52pt pill. The "ESC to cancel" badge downsizes to `.font(.caption2)` (11pt) from the existing `.font(.caption2)` — no change there.

## Color

All colors use existing `DesignSystem.Colors` tokens. No new color values are introduced in Phase 3.

| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | `DesignSystem.Colors.deepNavy` | #0F172A | Pill background at 0.92 opacity |
| Secondary (30%) | `DesignSystem.Colors.softBlueWhite` | #E0F2FE | Status label text, checkmark icon (done state) |
| Accent (10%) | `DesignSystem.Colors.vibrantBlue` | #2563EB | Waveform Canvas bars (recording), spinner arc stroke |
| Accent gradient | `DesignSystem.Colors.accentGradient` | vibrantBlue → teal | Transcribing spinner arc only |
| Idle icon | `DesignSystem.Colors.softBlueWhite` @ 50% opacity | #E0F2FE at 0.5 | Dim mic SF Symbol (idle state) |
| Destructive | `Color.red` | system red | Not used in this phase |

**Accent reserved for:** waveform Canvas bars (recording state only), transcribing spinner arc stroke. Not applied to pill background, text, or idle/done states.

**Pill background:** `Capsule().fill(DesignSystem.Colors.deepNavy.opacity(0.92))` — replaces `.ultraThinMaterial`. Rationale: ultraThinMaterial washes out the blue palette in dark mode; deepNavy at 0.92 opacity renders consistently across light and dark system appearances.

**Drop shadow:** `.shadow(color: .black.opacity(0.35), radius: 12, x: 0, y: 6)` — on the Capsule background layer.

Source: RESEARCH.md Pattern 1, existing DesignSystem.swift.

## Four-State Visual Contract

Each state must be immediately identifiable without label text as the sole differentiator. Requirements: OVLAY-02.

| State | Background | Icon/Shape | Color Signal | Label | Visible to user? |
|-------|-----------|------------|--------------|-------|-----------------|
| idle | deepNavy @ 0.92 | `mic` SF Symbol (not filled), 16pt | softBlueWhite @ 50% opacity | none | No — window hidden at idle; idle branch exists for preview completeness only |
| recording | deepNavy @ 0.92 | 12pt red dot + 32pt pulsing red halo + Canvas waveform | Red dot + vibrantBlue waveform | "Recording..." 13pt semibold softBlueWhite | Yes — window shown on hotkey down |
| transcribing | deepNavy @ 0.92 | 24pt arc spinner (trim 0→0.7, lineWidth 3) | accentGradient stroke | "Transcribing..." 13pt semibold softBlueWhite | Yes — immediately after key release |
| done | deepNavy @ 0.92 | `checkmark.circle.fill` SF Symbol, 20pt | softBlueWhite icon | none (no label) | Yes — 0.8s flash before window hides |

**Recording dot pulse:** Scale animation from 1.0 → 1.3, easeInOut 0.8s, repeatForever autoreverses: true. Phase-gated via `onChange(of: appState.phase)` pattern from FNDTN-04 (start on `.recording`, stop on any other phase with `withAnimation(.linear(duration: 0))`).

**Transcribing spinner:** Arc trim animation, rotation 0 → 360, linear 1s, repeatForever autoreverses: false. Phase-gated via same FNDTN-04 pattern (stop explicitly in `onChange` when phase exits `.transcribing`).

**Done state:** `checkmark.circle.fill` at 20pt in softBlueWhite. No green tint introduced — stays within the declared blue palette. Window hides at t+0.8s via `DispatchQueue.main.asyncAfter`. Source: RESEARCH.md open question 2 resolved in favor of softBlueWhite + checkmark, no new green token.

## Animation Contract

All animation values are prescriptive.

| Animation | Value | Applied To | Source |
|-----------|-------|------------|--------|
| State transition | `.spring(duration: 0.35, bounce: 0.1)` | ZStack `.animation(_:value: appState.phase)` | RESEARCH.md Pattern 3 |
| Transition type | `.opacity.combined(with: .scale(scale: 0.92))` | Each state branch `.transition()` modifier | RESEARCH.md Pattern 3 |
| Recording dot pulse | `.easeInOut(duration: 0.8).repeatForever(autoreverses: true)` | scaleEffect on outer circle | default — harmonious with spring duration |
| Transcribing spinner | `.linear(duration: 1).repeatForever(autoreverses: false)` | rotationEffect on arc | Existing pattern from TranscribingView.swift |
| Done flash duration | 0.8s (asyncAfter delay before hideRecordingOverlay()) | AppDelegate timing | RESEARCH.md Pattern 5 |
| Idle transition | 1.5s (asyncAfter delay before phase = .idle) | AppDelegate timing | Existing — unchanged |

**ZIndex:** Active state branch receives `zIndex(1)`. Prevents crossfade drawing-order artifacts where the exiting view renders above the entering view during the opacity transition. Source: RESEARCH.md Pattern 3 pitfall note.

## Copywriting Contract

The pill uses minimal copy. Labels must not wrap — the pill width is fixed at 280pt.

| Element | Copy | State | Font |
|---------|------|-------|------|
| Recording label | "Recording..." | recording | 13pt semibold (`.font(.subheadline).fontWeight(.semibold)`) |
| Transcribing label | "Transcribing..." | transcribing | 13pt semibold |
| Cancel hint badge | "ESC to cancel" | recording | 11pt regular (`.font(.caption2)`) |
| Done icon | *(no label — checkmark icon only)* | done | n/a |
| Idle | *(no label — dim mic icon only)* | idle | n/a |

**Primary CTA for this phase:** None — the overlay is display-only (`ignoresMouseEvents = true`). The Escape key cancel is an existing AppDelegate hotkey intercept, not a button.

**Empty state:** Not applicable — the overlay only appears when recording starts. There is no empty-data condition.

**Error state:** Not surfaced in the overlay. If transcription fails, the overlay hides at the done-state flash; the error is handled silently at the AppDelegate level (existing behavior, out of scope for Phase 3 UI).

**Destructive confirmation:** None in this phase. The overlay has no destructive actions. ESC-to-cancel is a hotkey intercept, not a button with confirmation.

Source: Existing RecordingOverlayView.swift copy carried forward with font size adjustments only.
