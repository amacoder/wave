**1. Add two instance properties** (near the other `var` declarations at the top of the class, around line 20):

```swift
private var recordingStartTime: Date?
```

**2. Capture metadata in startRecording()** — add two lines AFTER `appState.phase = .recording` (line ~223) and BEFORE `showRecordingOverlay()`:

```swift
recordingStartTime = Date()

```

This captures the source app at recording START per D-03/D-04 and RESEARCH Pattern 6. The user may switch apps during the 2-4s Whisper API call, so capturing early is the only reliable approach.

**3. Insert SwiftData save in transcribe()** — add a save block AFTER `var finalText` is computed (after the `if appState.smartCleanup` block, line ~308) and BEFORE the `await MainActor.run` block (line ~310). Insert this code:

```swift
// Save transcription to SwiftData (D-01: always save, even if paste fails — D-02)
let wordCount = finalText.split(separator: " ").count
let duration = recordingStartTime.map { Date().timeIntervalSince($0) } ?? 0
let sourceApp = recordingSourceApp

if let container = modelContainer {
    let bgContext = ModelContext(container)
    let entry = TranscriptionEntry(
        rawText: transcription,
        cleanedText: finalText,
        durationSeconds: duration,
        wordCount: wordCount,
        sourceAppName: sourceApp
    )
    bgContext.insert(entry)
    try? bgContext.save()
}

// Reset recording metadata
recordingStartTime = nil
recordingSourceApp = nil
```

Note: This save runs on the existing async context of transcribe() — it does NOT need Task.detached because transcribe() is already async. Use a separate `ModelContext(container)` (not the main view context) to avoid threading issues. The save happens BEFORE the MainActor.run paste block, so it persists even if paste fails (D-02).

**4. Add 90-day retention cleanup in applicationDidFinishLaunching()** — add at the END of the method, after all existing setup code:

```swift
// 90-day retention cleanup (D-17: silent, no notification)
if let container = modelContainer {
    Task {
        let context = ModelContext(container)
        let cutoff = Calendar.current.date(byAdding: .day, value: -90, to: Date())!
        let predicate = #Predicate<TranscriptionEntry> { $0.timestamp < cutoff }
        try? context.delete(model: TranscriptionEntry.self, where: predicate)
        try? context.save()
    }
}
```

Note: modelContainer may be nil at applicationDidFinishLaunching time because FlowSpeechApp.init() runs after. // In applicationDidFinishLaunching, at the end:
```swift
DispatchQueue.main.async { [weak self] in
```

```swift
private func cleanupOldEntries() {
```
