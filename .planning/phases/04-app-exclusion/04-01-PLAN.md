**InstalledApp struct** (nested inside AppExclusionService):
```swift
struct InstalledApp: Identifiable, Comparable {
    let id: String            // bundle ID
    let name: String
    let icon: NSImage
    static func < (lhs: InstalledApp, rhs: InstalledApp) -> Bool { lhs.name.localizedCaseInsensitiveCompare(rhs.name) == .orderedAscending }
```

**Published properties:**

- `@Published var autoSuppressFullscreen: Bool` -- with `didSet { UserDefaults.standard.set(autoSuppressFullscreen, forKey: "autoSuppressFullscreen") }`
- `@Published var installedApps: [InstalledApp] = []`

**Private property:** `private var metadataQuery: NSMetadataQuery?`

**Default bundle IDs constant:**
```swift
static let defaultExcludedBundleIDs: Set<String> = [
    "com.riotgames.LeagueofLegends",
    "com.riotgames.LeagueofLegends.LeagueClientUx"
]
```

**init():**
- Check `UserDefaults.standard.object(forKey: "excludedBundleIDs")` -- if nil (first launch), set `excludedBundleIDs = Self.defaultExcludedBundleIDs` and call `persist()`
- If not nil, load via `UserDefaults.standard.stringArray(forKey: "excludedBundleIDs") ?? []` and convert to Set
- Load `autoSuppressFullscreen` from UserDefaults with `object(forKey:)` nil-coalescing to `true` (default on)

**shouldSuppressHotkey() -> Bool:**
1. Get `NSWorkspace.shared.frontmostApplication` and its `bundleIdentifier` -- return false if either is nil
2. If `excludedBundleIDs.contains(bundleID)` return true
3. If `autoSuppressFullscreen` is true, return `frontmostAppIsFullscreenOrBorderless(pid: frontmost.processIdentifier)`
4. Otherwise return false

**frontmostAppIsFullscreenOrBorderless(pid: pid_t) -> Bool (private):**
1. Use `CGWindowListCopyWindowInfo([.optionOnScreenOnly, .excludeDesktopElements], kCGNullWindowID)` cast as `[[String: Any]]`
2. Get `NSScreen.main?.frame` for screen dimensions
3. Iterate windows: filter by `kCGWindowOwnerPID` matching `pid`
4. Extract bounds dict from `kCGWindowBounds` key, get Width/Height/X/Y as CGFloat
5. Check full coverage: `abs(w - screenFrame.width) < 2 && abs(h - screenFrame.height) < 2` -> return true
6. Check near-coverage: `w >= screenFrame.width * 0.99 && h >= screenFrame.height * 0.95` -> return true
7. If no window matched, return false

**startInstalledAppsQuery():**
1. Create `NSMetadataQuery()`
2. Set `searchScopes` to `["/Applications", NSHomeDirectory() + "/Applications"]`
3. Set `predicate` to `NSPredicate(format: "kMDItemContentType == 'com.apple.application-bundle'")`
4. Set `sortDescriptors` to `[NSSortDescriptor(key: kMDItemDisplayName as String, ascending: true)]`
5. Add observer for `.NSMetadataQueryDidFinishGathering` targeting `queryDidFinish(_:)`
6. Store query in `metadataQuery` property
7. Call `query.start()`

**queryDidFinish(_ note: Notification) (private @objc):**
1. Stop the query
2. Iterate results: extract path from `kMDItemPath`, create URL, get Bundle, get bundleIdentifier
3. Get display name from `kMDItemDisplayName` with fallback to filename
4. Get icon via `NSWorkspace.shared.icon(forFile: path)`
5. Skip entries where bundleIdentifier is nil
6. Dispatch to main: set `installedApps = apps.sorted()`

**persist() (private):**
- `UserDefaults.standard.set(Array(excludedBundleIDs), forKey: "excludedBundleIDs")`
